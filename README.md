# Hospital Patient Analysis Dashboard

## Project Overview
This project analyzes hospital patient data using Power BI to understand admission trends, patient distribution, and hospital workload.

## Tools Used
- Power BI
- Power Query
- Data Visualization
- Data Analysis

## Key Metrics
- Total Patients: 150K
- Total Hospitals: 32
- Average Admission Deposit: 4988
- Average Visitors: 3.24

## Key Insights
- Trauma admissions represent the largest portion of hospital cases.
- The Gynecology department handles the highest patient volume.
- Moderate severity cases dominate hospital admissions.
- Patient demand is concentrated in a few major cities.
